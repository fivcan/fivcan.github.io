  • 注意：父进程结束时，子进程并不会随父进程立刻结束。同样，父进程不会等待子进程执行完。
  • 注意：os.fork()无法在windows上运行。

套路4：multiprocessing.Pool 进程池

  • 对Pool对象调用join()方法会等待所有子进程执行完毕。调用join()之前必须先调用close()，调用close()之后就不能继续添加新的Process了。
